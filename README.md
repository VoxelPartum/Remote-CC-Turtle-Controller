# REMOTE CONTROL COMPUTERCRAFT TURTLE
(Requires _Node.js_ & _CC:Tweaked_ v1.113.1)

**Preparation:**
Make sure [Node.js](https://nodejs.org/en) and [CC:Tweaked](https://www.curseforge.com/minecraft/mc-mods/cc-tweaked) are installed.
Before installing any programs, construct a constellation as directed [here](https://tweaked.cc/guide/gps_setup.html).

**USAGE:**
1. Open the Nodejs server in a command window with "node ./server.js".
2. Install and run the "rc_controller.lua" program on an _advanced_ turtle, equipped with a ender modem.
  - (Note: Make sure to load the inventory with fuel beforehand.)
3. Open the "index.html" in your browser.

**Instructions:**
Simply type in the coordinates you desire to have the turtle move to, then press the send button.
The display on the right displays the 9 blocks underneath the turtle, hover your mouse over any one of them to determine what block it is, if it has seen it before.
